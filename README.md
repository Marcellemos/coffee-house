#  Lemos Coffee House

Site simples de apresentação para uma cafeteria — template front-end estático.

## Descrição

Projeto contendo uma landing page com HTML, CSS e JavaScript para demonstrar um layout de cafeteria, galeria de imagens e reprodução de vídeos.

## Estrutura do projeto

- `index.html` — página principal
- `style.css` — estilos
- `script.js` — scripts JavaScript
- `img/` — imagens do projeto
- `videos/` — vídeos usados na página

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Como usar

1. Abrir `index.html` no navegador (duplo clique ou via servidor local).
2. Para experiência de desenvolvimento, recomendo usar a extensão Live Server do VS Code ou um servidor simples:


## Desenvolvimento

- Edite `style.css` para ajustar estilos.
- Edite `script.js` para alterar comportamentos/interações.
- Coloque imagens em `img/` e vídeos em `videos/`.
